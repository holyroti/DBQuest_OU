Lets get started setting up this project!

1. Install angular
2. Install nodejs
3. Update the node modules from the root folder ``npm install``

4. Run the code! from the root folder open in a terminal: ``ng serve``
5. To start the backend, after configuring the database files + .env files (in the backend folder) execute in a terminal: ``node \backend\server.js``

6. The angular application should compile

7. Do notice that no data will be loaded, since your database will be empty! Please refer to the SQL repositories containing the SQL scripts and feel free to change them according to your educational purpose.
